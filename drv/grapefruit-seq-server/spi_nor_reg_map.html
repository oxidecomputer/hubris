Name Bit(s) Access Reset Description
sp5_owns_flash 31 rw 0x0 Set to one to allow eSPI <-> SP5 interface read from flash. Hubris should not attempt any flash accesses with this bit set. This bit also enforces the flash offset in

Name Bit(s) Access Reset Description
tx_used_wds 30..24 rw 0x0 Show used FIFO words in TX_FIFO (word = 32bits/4 bytes), max of 256 bytes, so 64 words
tx_full 23 rw 0x0 Set to one to 1 when TX FIFO is full
tx_empty 22 rw 0x0 Set to one to 1 when TX FIFO is empty
rx_used_wds 14..8 rw 0x0 Show used FIFO words in RX_FIFO (word = 32bits/4 bytes), max of 256 bytes, so 64 words
rx_full 7 rw 0x0 Set to one to 1 when RX FIFO is full
rx_empty 6 rw 0x0 Set to one to 1 when RX FIFO is empty
busy 0 rw 0x0 Set to one to 1 when hw is running the spi transaction, no new transactions may be issued until it is finished. Technically represents cs_n being low.
