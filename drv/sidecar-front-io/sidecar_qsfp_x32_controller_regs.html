Name Bit(s) Access Reset Description

 VSC8562_PHY_CTRL  
0x101
 Control bits related to VSC8562 (valid on FPGA1 only)
 7..0 RSVD RESET REFCLK_EN CLEAR_POWER_FAULT COMA_MODE EN
Name Bit(s) Access Reset Description
RESET 4 rw 0x0 Software control for the FPGA1_TO_PHY_RESET_L net. Only effective after PHY initialization.
REFCLK_EN 3 rw 0x1 Software control for the FPGA1_TO_PHY_REFCLK_EN net. Only effective after PHY initialization.
CLEAR_POWER_FAULT 2 rw 0x0 Setting this bit to 1 will clear the timed out state of the V1P0 and V2P5 rail controllers, allowing PHY power sequencing to be attempted again.
COMA_MODE 1 rw 0x1 Software control for the FPGA1_TO_PHY_COMA_MODE net. Only effective after PHY initialization.
EN 0 rw 0x1 Enable for the VSC8562 power up state machine
